## Se mejoraron lso siguientes temas.

* Sepuede abrir mas de una vez, cada opción del menú.

* Muestra el ID de cada ipción, en el menú, sirve para hacer soporte mas rápido.

* Una opción de menú, puede ser de tipo Link, por ejemplo, para ejecutar un reporte en SSRS.

* Una opción, permite eliminar muchos registros a la vez.

* En modo de edición o formulario, se puede ajustar el ancho, estirando la pantalla con el mouse.

* En una columna de tipo texto, se puede adjuntar un path a un documento de la red, o bien del equipo local, y con un clic, se abre automáticamente.
